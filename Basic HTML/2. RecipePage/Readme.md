# Delicious Pancakes 🥞

A modern, clean webpage that showcases a fluffy, golden pancake recipe.  
Built as part of **FreeCodeCamp Responsive Web Design** exercises, optimized for portfolio presentation.

## Features

- Semantic HTML5 structure (`<main>`, `<section>`, `<ul>`, `<ol>`, `<img>`, `<header>`, `<footer>`)  
- Responsive layout with hover effects  
- Gradient-based modern styling  
- Clean typography for readability  
- External links open in new tabs for user safety  

## Technologies Used

- HTML5
- CSS3
