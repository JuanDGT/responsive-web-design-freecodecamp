# 🎵 freeCodeCamp Tunes

This is a **modern HTML music player** built as part of the freeCodeCamp Responsive Web Design projects.

## 🚀 Live Demo
Check it out on CodePen:  
[View Music Player](https://codepen.io/Juan-Diego-Gutierrez-Torres/pen/ZYpjmzp)

## 📌 Project Description
A simple and professional-looking music player that showcases three curated tracks by Quincy Larson:

- "Can't Stay Down"  
- "Cruising for a Musing"  
- "Scratching the Surface"  

Each song features **playback controls, looping, and optional dark mode**.

## ✨ Features
- 🎶 Multiple audio tracks  
- 🔊 Audio controls: play, pause, seek, volume  
- 🔁 Looping for continuous playback  
- 🌙 Dark mode toggle with JavaScript  
- 📱 Responsive and modern layout with CSS styling  
- ✨ Hover effects and card-style design  

## 🛠 Technologies Used
- HTML5  
- CSS3 (Flexbox, gradients, shadows)  
- Vanilla JavaScript
