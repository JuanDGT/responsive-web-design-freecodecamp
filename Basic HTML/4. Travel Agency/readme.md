## 📌 Description

A modern and responsive travel agency landing page built with **HTML, CSS, and JavaScript**.
It showcases travel packages and top destinations with a clean and interactive design.

## ✨ Features

* 🌐 Semantic HTML structure
* 🎨 Modern UI with CSS styling
* 📱 Responsive layout (Flexbox)
* 🌙 Dark mode toggle (JavaScript)
* 🖼️ Image gallery with hover effects

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
