# RMC Bookstore 📚

A modern, responsive online bookstore page built as part of **FreeCodeCamp Responsive Web Design** exercises.

## Features

- Semantic HTML5 (`<main>`, `<section>`, `<header>`, `<footer>`)  
- Responsive card grid layout for books  
- Hover effects on book cards  
- Modern gradients and typography for a polished look  
- Interactive buttons with smooth hover animations  

## Technologies Used

- HTML5
- CSS3
