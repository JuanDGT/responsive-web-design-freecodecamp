# CatPhotoApp 🐱

CatPhotoApp is an interactive, modern webpage that showcases adorable cat photos and cat lists.  
This project was built as part of **FreeCodeCamp HTML & CSS exercises** and is structured for professional portfolio presentation.

## Features

- Fully semantic HTML5 (`<main>`, `<section>`, `<figure>`, `<figcaption>`, `<header>`, `<footer>`)
- Modern, responsive layout using CSS Grid
- Hover effects on sections and images
- Clean typography and gradient-based design
- External links open in new tabs for safe browsing

## Technologies Used

- HTML5
- CSS3
